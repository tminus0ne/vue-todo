# vue-todo

A simple Vue 3 ToDo app.
Developed using Vue Router, Vuex and Vue Composition API.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
